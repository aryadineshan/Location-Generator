# Location-Generator
An Android application developed using Android Studio, which converts the user-entered location into its geographic coordinates using Androidâ€™s Geocoder class and displays the location on Google Maps. 

## Features
- Geocoding: Converts any user-entered address to its geographical coordinates.
- Show on Map: Opens the location in Google Maps or your default browser.


## Tech Stack
- Java
- Android SDK
- Geocoder API
- Android Studio 

## How It Works
1. **Enter an address** in the input field.
2. Tap **Geocode**.
3. The app displays the latitude and longitude.


## Permissions
This app uses:
- `INTERNET`
- `ACCESS_FINE_LOCATION`
- `ACCESS_COARSE_LOCATION`
