# Location-Generator
An Android application developed using Android Studio, which converts the user-entered location into its geographic coordinates using Androidâ€™s Geocoder class. 

## How It Works

1. **Enter an address** in the input field.
2. Tap **Geocode**.
3. The app displays the latitude and longitude.

## Tech Stack

- Java
- Android SDK
- Geocoder API

## Permissions
This app uses:
- `INTERNET`
- `ACCESS_FINE_LOCATION`
- `ACCESS_COARSE_LOCATION`
